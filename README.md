# Chai Practice

A practice app for learners of conversational Persian - vocabulary,
phrase and script games with a daily practice session and gentle spaced repetition.

Built with content from Chai and Conversation. A concept prototype, not an official Chai and Conversation product. Content is owned by Chai & Conversation. Content is subject to copyright.
